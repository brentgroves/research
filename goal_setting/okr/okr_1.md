# OKR Template

OKRs are made up of two components:  

- **Objective:** what you’re trying to accomplish
- **Key results:** how you’ll measure whether you achieve the objective

## Key Result format

We will `objective` as measured by `these key results`.

```yaml
objectives:
  - linamar: Customer – Supplier of Choice
  - structures: Automate the JT Fronts performance index calculation process.
key_results: We will automate the JT Fronts performance index calculation as measured by:
  enable data collection

```

Objective: Customer – Supplier of Choice
KR1: 100% confidence in Linamar's ability to produce good parts.
KR2: Achieve JT Fronts performance index above 1.3.
KR3: Achieve 1% scrap rate.
